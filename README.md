# gsc_sitepackage
Typo3 Sitepackage for GSC Muenchen
 
This Sitepackages brings the look&feel of our TYPO3 instance. It contains all necessary (S)CSS, JavaScript, Icons, Images, Logos, modified Templates and Partials and all the other awesome stuff you can see on our website.
Please ensure that [Bootstrap](https://getbootstrap.com/) framework and jQuery is enabled.

## Requirements

- TYPO3 >= 12.4
- ext:bootstrap_package >= 15.0
- ext:lib_jquery >= 5.0

## Installation

- Install the extension by using the extension manager or use composer
- Include the static TypoScript of the extension

## Usage

- should work out of the box
- if encountering problems, it might be useful to install the extension first and then create pages. Don't forget to add "page = PAGE" to your TypoScript setup!

## Bugs and Known Issues
If you find a bug, it would be nice if you add an issue on [Github](https://github.com/GSC-Munchen/gsc_sitepackage/issues).
